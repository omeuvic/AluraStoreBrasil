# Análise das Lojas do Senhor João

## Sobre o projeto

Este projeto foi feito em Python usando o Google Colab para analisar os dados de vendas de quatro lojas do Senhor João.

O objetivo foi entender o desempenho das lojas, como faturamento, produtos mais vendidos, avaliações dos clientes e valor do frete.

---

## O que foi analisado

- Faturamento total de cada loja  
- Produto mais vendido  
- Produto menos vendido  
- Categorias mais vendidas  
- Média de avaliação dos clientes  
- Frete médio  
- Gráficos usando matplotlib  

---

## Ferramentas utilizadas

- Python  
- Google Colab  
- pandas  
- matplotlib  

---

## Como executar

1. Abrir o Google Colab  
2. Fazer upload dos arquivos CSV das lojas  
3. Executar o código passo a passo  

---

## Conclusão

Com a análise, foi possível identificar o desempenho de cada loja e entender qual loja teve menor desempenho. Isso ajuda na tomada de decisão e melhora da estratégia de vendas.
